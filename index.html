关于JQuery中 和.get()和 .post()和$.ajax()的区别和使用

发表于 2017-10-08 | 分类于 技术
jQuery get() 和 post() 方法用于通过 HTTP GET 或 POST 请求从服务器请求数据。

GET - 从指定的资源请求数据
POST - 向指定的资源提交要处理的数据
GET 基本上用于从服务器获得（取回）数据。注释：GET 方法可能返回缓存数据。
POST 也可用于从服务器获取数据。POST 方法不会缓存数据，并且常用于连同请求一起发送数据。
