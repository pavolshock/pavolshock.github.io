浏览器对象

发表于 2017-11-01 | 分类于 技术

浏览器对象是指：除去工具栏，导航栏，菜单栏，边框等占位元素后的，用于显示网页的净宽高。

window对象充当全局作用域，而且表示浏览器窗口。
window对象有innerwidth和innerheight 属性，可以回去浏览器窗口内部的宽高。

navigator对象表示浏览器的信息，最常用的属性包括：

. navigator.appName:浏览器名称；
. navigator.appVersion:浏览器版本；
. navigator.;language：浏览器设置语言；
. navigator.platform:操作系统类型；
. navigator.userAgent: 浏览器设定的user － agent 字符串。

navigator 的信息可以很容易滴被用户修改，所以javascript读取的值不一定是正确的，不要用navigator去判断浏览器版本正确的方法是充分利用javascript对不存在属性返回undefinded的属性，直接用短路运算｜｜计算：
var width = window.innerWidth || document.body.clientWidth;

screen 对象表示屏幕的信息，常用的属性有：
. screen.width:屏幕宽度，一像素为单位；
. screen.height: 屏幕高度，以像素位单位；
. screen.colorDepth: 返回颜色位数，如8、16、24.

‘use strict’;
alert (’Screen size = ‘ +screen.width + ‘x’ + screen.height);

location
location 对象表示当前页面的url信息。例如：
一个完整的url地址：
http://www.example.com:8080/path/index.html?a =1&b =2#TOP
可以用location.href 获取。要获取url各个部分的值，可以这么写；
location.protocol; //‘http’
localhost.host; //‘www.example.com’
localhost.port;//‘8080’
localhost.pathname; //‘path/index.html’
localhost.search;//‘?a =1&b =2’
localhost.hash;//’TOP’
要加载一个新页面可以调用localhost.assign().如果要重新加载当前页面，调用localhost.reload()方法非常方便

‘use strict’;
if(confirm(‘重新加载当前页’ + localhost.href + ‘?’)) {
location.reload();
}else {
localhost.assign(‘/discuss’);//设置一个新的url地址
}

document

document 对象表示当前页面，由于html在浏览器中以DOM形式表示为树形结构，document
对象就是整个DOM树的根节点。
document的title属性是从html文档中的xxx读取的，但是可以动态改变。
‘use strict’；
document.title = ‘努力学习Javascript中’;
请观察浏览器窗口标题的变化。
要查DOM树的某个节点，需要从document对象开始查找，最常用的查找是根据ID和Tag Name。

用document 对象提供的getelementbyid() getelementsbytagname() 可以按
id回去一个DOM节点和按tag名称获得一组DOM节点；
drinks ＝ document.getElementByTagName(‘dt’);
s = ‘提供服务的是：’;

document 对象还有一个cookie属性，可以获取当前野 main的cookie

cookie 是由服务器发送的key－value标识符。因为http是无状态协议，但是服务器窑区分到底是哪个用户发送过来的请求，就可以用cookie来区分，当一个用户成功登陆后，服务器发送一个cookie给浏览器，

javascript 可以通过document.cookie读区到当前页面的cookie；
document.cookie:可以读取当前页面的cookie信息，第三方引入也是可以的
所以说是非常危险的，有可能第三方恶意代码，为乐解决这个问题，服务器在设置cookie时可以使用httponly，设定了httponly的cookie将不能被javascript读取，这个行为由浏览器实现，主流浏览器均支持，
为了确保安全，服务器端在设置cookie时，应该始终坚持使用httponly。
history对象 保存了浏览器的历史纪录，javascript可以调用history对象和back（）
或者forward（），相当于第几了浏览器的前进与后退

这个对象历史遗留对象，由于ajax和页面交互，简单粗暴的调用hitstory。bakc（）可能让用户感觉非常封路

浏览器对象有： windows对象 ，location对象，screen对象，document对象，navigator对象。
